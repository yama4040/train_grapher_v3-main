# train-grapher-v3
train grapher v2のリファクタリングバージョン。
webに組み込むバックエンドとも共通化。

# 環境構築
以下のソフトウェアをダウンロード、インストールする。

* git (バージョン管理)
* uv (pythonパッケージ管理ソフトウェア)
* vscode (エディタ)

本リポジトリをクローンする。
```bash
git clone [リポジトリURL] 
```

vscodeでクローンしたリポジトリを開く。

uvを用いてpythonライブラリをインストールする。
```bash
uv sync
```

vscodeに以下の拡張機能を追加する。

* Ruff

# 動かし方
uvが作成する仮想環境で実行する

```bash
uv run python ./src/train_grapher_v3
```
